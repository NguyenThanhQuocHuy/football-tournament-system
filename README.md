# Football Tournament Management System

## Description
Website quản lý giải đấu bóng đá phong trào/bán chuyên.
Hỗ trợ ban tổ chức quản lý đội bóng, cầu thủ, lịch thi đấu, kết quả và bảng xếp hạng.

## Features
- Quản lý giải đấu
- Quản lý đội bóng & cầu thủ
- Tạo lịch thi đấu tự động
- Nhập kết quả trận đấu
- Tính bảng xếp hạng tự động
- Tin tức & thông báo
- Phân quyền admin / đội / khán giả

## Tech Stack
- PHP (MVC)
- MySQL
- Bootstrap
- JavaScript
- XAMPP
## Installation
1. Clone project
2. Import database `tournament_db.sql`
3. Run on XAMPP
